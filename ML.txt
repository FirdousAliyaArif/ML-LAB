1. What is Machine Learning?
Machine Learning is a branch of Artificial Intelligence where computers learn patterns from data and make predictions without explicit programming.

2. Why is Machine Learning used?
To analyze large data and automatically make predictions or decisions.

3. What are the types of Machine Learning?

Supervised Learning

Unsupervised Learning

Reinforcement Learning

4. What is Supervised Learning?
A method where the model is trained using labeled data.

5. What is Unsupervised Learning?
Learning from unlabeled data to find hidden patterns.

6. What is Reinforcement Learning?
A learning method where an agent learns through rewards and penalties.

7. What is a dataset?
A collection of structured data used to train and test ML models.

8. What are features?
Input variables used to train the model.

9. What is a label?
The output variable that the model predicts.

10. What is training data?
Data used to train the machine learning model.

11. What is testing data?
Data used to evaluate model performance.

12. What is a model in ML?
A mathematical representation that learns patterns from data.

13. What is prediction?
The output generated by a trained model.

14. What is a target variable?
The variable that the model tries to predict.

15. What is data mining?
The process of discovering patterns from large datasets.

16. What is feature engineering?
The process of selecting or transforming features for better model performance.

17. What is pattern recognition?
Identifying patterns or regularities in data.

18. What is AI?
Artificial Intelligence is the field of creating machines that mimic human intelligence.

19. What is Deep Learning?
Deep Learning is a subset of ML using neural networks with many layers.

20. What is Big Data?
Extremely large datasets that require advanced tools to process.

21–40: Algorithms

21. What is Linear Regression?
Linear Regression predicts continuous values using a straight-line relationship.

22. What is Logistic Regression?
Logistic Regression is used for binary classification problems.

23. What is KNN?
K-Nearest Neighbors Algorithm classifies data based on the nearest data points.

24. What is a Decision Tree?
Decision Tree Learning is a model that splits data based on conditions.

25. What is Random Forest?
Random Forest is an ensemble of multiple decision trees.

26. What is Naive Bayes?
Naive Bayes classifier is based on Bayes theorem and assumes feature independence.

27. What is SVM?
Support Vector Machine is used for classification and regression tasks.

28. What is clustering?
Grouping similar data points together.

29. What is K-means clustering?
K-means clustering divides data into K groups based on similarity.

30. What is hierarchical clustering?
A clustering method that builds nested clusters.

31. What is ensemble learning?
Combining multiple models to improve accuracy.

32. What is boosting?
A method that improves weak learners sequentially.

33. What is bagging?
Training multiple models on random subsets of data.

34. What is gradient descent?
An optimization algorithm used to minimize loss.

35. What is loss function?
A function that measures model prediction error.

36. What is cost function?
Average loss over the dataset.

37. What is hypothesis in ML?
The function that maps inputs to outputs.

38. What is bias in ML?
Error caused by overly simple models.

39. What is variance in ML?
Error due to model sensitivity to training data.

40. What is bias-variance tradeoff?
Balancing bias and variance for better performance.

41–60: Model Evaluation

41. What is accuracy?
Ratio of correct predictions to total predictions.

42. What is precision?
Correct positive predictions divided by total predicted positives.

43. What is recall?
Correct positives divided by actual positives.

44. What is F1-score?
Harmonic mean of precision and recall.

45. What is a confusion matrix?
A table showing predicted vs actual values.

46. What is cross-validation?
Technique to evaluate model using multiple data splits.

47. What is k-fold cross validation?
Dataset divided into k parts for training/testing.

48. What is ROC curve?
Graph showing model performance at different thresholds.

49. What is AUC?
Area under ROC curve measuring classification performance.

50. What is mean squared error?
Average squared difference between actual and predicted values.

51. What is RMSE?
Square root of mean squared error.

52. What is MAE?
Mean absolute error.

53. What is validation set?
Data used to tune model parameters.

54. What is model evaluation?
Assessing how well a model performs.

55. What is prediction error?
Difference between predicted and actual value.

56. What is classification report?
Summary of precision, recall, and F1-score.

57. What is true positive?
Correct positive prediction.

58. What is false positive?
Incorrect positive prediction.

59. What is true negative?
Correct negative prediction.

60. What is false negative?
Incorrect negative prediction.

61–80: Data Preprocessing

61. What is data preprocessing?
Cleaning and preparing data before training.

62. What is data cleaning?
Removing errors and missing values.

63. What are missing values?
Data points that are empty or unknown.

64. How to handle missing data?
Delete rows or fill using mean/median/mode.

65. What is normalization?
Scaling data to range between 0 and 1.

66. What is standardization?
Transforming data to mean = 0 and standard deviation = 1.

67. What is feature scaling?
Adjusting feature values to similar ranges.

68. What is encoding?
Converting categorical data into numeric format.

69. What is label encoding?
Assigning numbers to categories.

70. What is one-hot encoding?
Creating binary columns for each category.

71. What is outlier?
An extreme value different from other observations.

72. What is dimensionality reduction?
Reducing number of features.

73. What is PCA?
Principal Component Analysis reduces feature dimensions.

74. What is feature selection?
Choosing important features for training.

75. What is data transformation?
Converting data into suitable format.

76. What is imbalance dataset?
Dataset where classes are uneven.

77. What is sampling?
Selecting subset of data.

78. What is oversampling?
Increasing minority class samples.

79. What is undersampling?
Reducing majority class samples.

80. What is data visualization?
Graphical representation of data.
